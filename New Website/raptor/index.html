How it works

Encryption Data Dispersal Recieval and Decryption

Encryption

The message you send is encrypted using the encryption RSA algorithm, making only the the designated reciever of the message to be able to decrypt the message.

The encrypted message is then sent out to be intercepted by the reciever.

Data Dispersal

Unlike many other security sytems that rely on a central system that keeps all data stored, which can potentially be breached and compromise privacy and security, Byte-Mail seeks to decentralize the entire procedure.

Byte-Mail quickly disperses encrypted data among a multitude of users on a network, making it impossible for outsiders to track and decrypt messages that you are sending

Recieval and Decryption

After traversing through a complex network, the encrypted data is stopped when it is sent to the designated reciepiant. All of this happens in a matter of seconds.

The reciever is then able to decrypt the the message using a private key to the encryption that they only know.
